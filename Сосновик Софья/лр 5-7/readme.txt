Вопросы для лабораторных работ в R-Studio

*Что такое RStudio? Ответ:RStudio - это интерфейс для работы с языком R.
*Что такое язык R? Ответ:R - это язык программирования для обработки данных.
*Какие типы данных есть в языке R? Ответ:Символьный, числовой, логический, комплексный.
*Как в языке R можно объединить несколько значений? Ответ:Можно объединить в вектор, массив, матрицу, список, таблицу.
*Как в языке R присвоить значение объекту?Ответ: С помощью оператора <-
*Как создать вектор?  Ответ: c(...)
*Как создать список? Ответ: list(...)
*Как создать массив?  Ответ:array(x, dim = )
*Как создать матрицу? Ответ: matrix(x, nrow = , ncol = )
*Перечислите основные математические функции. Ответ: max(x), min(x), range(x), sum(x), prod(x), mean(x), median(x), quantile(x, probs = ...), var(x), cov(x).
*Как посторить график? Ответ:plot(x, y).
*Как добавить к графику легенду? Ответ: legend(x, y, legend).
*Как можно задать цвет для графика или диаграммы? Ответ: С помощью параметра col
*Как считать данные из текстового файла? Ответ:Для этого используются три функции: read.table() (которая имеет два варианта: read.csv() и read.csv2()), scan() и read.fwf().
*Как сохранить текстовые данные? Ответ:Для сохранения данных используются write.table(), write.matrix().
*Какие есть  основные операции с матрицами? Ответ:Функции rbind() и cbind() объединяют матрицы относительно строк или столбцов, Транспонирование матрицы производится функцией t(); Функция diag() может использоваться, чтобы извлечь или изменить
диагональ матрицы, или для построения диагональной матрица; а также основные математические опрации +-*/
*Как получить один элемент трехмерного массива? Ответ: Указать в индексах все три размерности: array[2,1,3].
*Как добавить на корреляционное поле линию регрессии? Ответ: С помощью функции abline().
*Как построить модель линейной регресии? Ответ: С помощью функции lm().
*Что делает функция summary()? Ответ: Позволяет оценить адекватность модели с помощью F-статистики.
*Как определить корреляцию между независимым и зависимым факторами? Ответ: С помощью функции cor().
