Что такое MongoDb? Ответ: MongoDB реализует новый подход к построению баз данных, где нет таблиц, схем, запросов SQL, внешних ключей и многих других вещей, которые присущи объектно-реляционным базам данных.
Какие форматы данных есть в MongoDb? Ответ: JSON и BSON
Что такое коллекция? Ответ: это аналог таблиц в обычных sql базах данных
Какой командой можно установить текущую бд? Ответ: > use db_name
Как можно вставить данные? Ответ: 
    insertOne(): добавляет один документ
    insertMany(): добавляет несколько документов
    insert(): может добавлять как один, так и несколько документов
Как загрузить файл? Ответ: > load("file_path")
Как выглядит аналог SELECT * FROM Table в MongoDb? Ответ: > db.collection_name.find()
Что такое _id ? Ответ: Это уникальное 12-байтовое шестнадцатеричное число, которое либо вы указываетет сами, либо его создает система.
