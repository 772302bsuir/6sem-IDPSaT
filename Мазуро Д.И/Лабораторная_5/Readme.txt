1. seq() - функция, генерирующая последовательность чисел
Если Я хочу создать вектор на 20 элементов, то пишу seq(20)
Можно присваивать свой вектор, например : v<- c(1,2,3)
2. <- оператор присваивания
myset <- seq(20) - определить myset как вектор из 20 элементов
3. Вывод в консоль:
- курсор на нужную строку и нажать run (ctrl + Enter)
- выделить необходимые действия и нажать run (ctrl + Enter)
4. Основные функции
sum() - сумма элементов в векторе
max() - максимальное значение
min() - минимальное значение
prod() - возвращает произведение всех значений внутри
length() - количество элементов
mean() - среднее арифметическое
var() -  возвращает дисперсию
sort() - сортирует элементы по возрастанию 
summary() - функция для содзания итоговых результатов. Вызывает методы, которые зависящие от класса первого аргумента
