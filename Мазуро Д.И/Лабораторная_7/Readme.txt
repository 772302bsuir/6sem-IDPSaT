1. Для построения регрессионного поля используем  plot()
2. cor() - используем для нахождения корреляции между независимыми и зависимыми факторами
3. abline() - эта функция добавляет одну или несколько прямых линий через текущий график
4. lm()- строит модель линейной регрессии
5. attributes() - используется, чтобы увидеть все атрибуты объекта
6. residuals() - позвоялет получить ряд остатков
7. summary() - оценивает адекватность модели, опираясь на F-statistic
8. Data mining - это автоматизированный поиск данных, основанный 
на анализе огромных массивов информации
