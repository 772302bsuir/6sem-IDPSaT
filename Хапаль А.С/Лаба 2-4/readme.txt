1)Что за функция Show Me?
Это функция, которую можно использовать для применения требуемого представления к существующим данным на листе. Эти виды могут быть круговой диаграммой, точечной диаграммой или линейной диаграммой.
2)Как показать связь между двумя полями с помощью Show Me?
Выберать два поля для анализа, удерживая клавишу управления, а затем нажать на кнопку Show Me и выбрать график.
3)Что за функция Connect Live? 
Функция Connect Live используется для анализа данных в реальном времени.
4)Как получить подробное описание линии тренда? 
Для получения подробного описания линии тренда необходимо щелкнуть правой кнопкой мыши на график и выбрать "Описать линию тренда". 
Подробное описание показывает нам коэффициенты, значение перехвата и уравнение.
